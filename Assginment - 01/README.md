## 📖 Project Overview

KnowledgeVault is a simple yet visually appealing landing page designed to showcase a modern web layout. The project emphasizes:

* Semantic HTML structure
* Clean and maintainable CSS
* Responsive layout design
* Proper spacing and alignment
* User-friendly interface
* Professional UI implementation based on Figma design
